# css

# 一.清除浮动的主要方法及其原理

  • 不推荐的方法: 1.overflow: hidden 2.增加额外标签，然后添加属性clear: both 3.给父元素添加高度 4.给所有元素添加浮动

  • 推荐方法：after伪类清除

.clearfloat:after{display:block;clear:both;content:"";visibility:hidden;height:0}
.clearfloat{zoom:1} /兼容ie6,7/

# 二.图片1px间隙

  • 原因:img标签为行内元素，默认对其方式为vertical-align: baseline(基线对齐) 。图片默认的垂直对齐方式是基线，基线的位置与字体相关，
  • 解决办法：1)将display属性改成块2）vertical-align属性设置成bottom

# 四.margin塌陷

  • 原因: margin在垂直方向发生合并
  • 发生合并的两个条件 1）块级元素，但不包含浮动和绝对定位元素 2）在默认文档流下只出现在垂直方向
  • margin合并的三种场景 1）相邻的兄弟元素 2）父级和第一个/最后一个子元素 3）空块级元素margin合并
  • 解决方案 1）父级设置为块级格式化上下文元素 2）父元素设置border-top/bottom值
    3）父元素设置padding-top/bottom值 4）父元素设置高度
